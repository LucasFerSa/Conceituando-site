Dicionário

  • Back-End: É a parte da programação que o usuário não vê, é onde todo o processamento e armazenamento dos dados acontece. Um exemplo é os dados que o site recebe cada vez que você curte algo, isso é armazenado no back-end.
  • Front-End: Opostamente do back-end, o front-end é a parte que o usuário enxerga e tem acesso. Basicamente o front-end possui o HTML, CSS e JavaScript. Um exemplo é quando você curte um post no instagram e o coração muda para a cor vermelha.
  • Github: É um site os programadores e desenvolvedores postam seus trabalhos, acessam trabalhos de outros e conseguem interagir com os mesmos. É como se fosse uma rede social para essa área.
  • Git: É uma ferramenta utilizada para um melhor controle do seu código, podendo ser usado para não alterar o principal do seu site caso você cometa um erro, ou para que várias pessoas trabalhem no mesmo projeto sem interferir no código um do outro
  • Linguagem: É o que usamos para traduzir da nossa lingua o que queremos que o computador faça para a "lingua" que o computador entende.

Linguagens

HTML

Não é linguagem de programação, e sim de marcação. Atuada no Front-end, sua função é marcar o que cada conteúdo é na página (Título, parágrafo, lista...)

CSS

Atuada no Front-end ,também não considerado linguagem de programação, e sim de estilização. É com o CSS que você deixa sua página bonitinha

JavaScript

Linguagem de programação muito diversa pois se é usada tanto no Front-End(Responsável pela interação que o usuário tem com a página) e no Back-End.

Python

Linguagem Back-End,é utilizada principalmente para aprendizado de máquina e para manipulação de dados, é famosa por ser simples e de fácil entendimento

C

É uma ótima linguagem para sistemas operacionais, é encontrada em diversos jogos e softwares. Também sendo linguagem Back-End

R

Linguagem principalmente utilizada por estatísticos, pois é usada para análise de dados e desenvolvimento de softwares analíticos

Lua

Tem características que a torna ideal para configurar e automatizar. Não é conhecida, mas merece seu espaço pois é uma linguagem Brasileira
